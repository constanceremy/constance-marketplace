Review the pull request on the current branch.

Check for:
- Logic errors or unintended side effects
- Security issues (injection, unvalidated input, exposed secrets)
- Missing or inadequate test coverage
- Consistency with the existing code style

Report findings grouped by severity: critical, warning, suggestion.
